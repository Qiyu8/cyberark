# 南北战争爆发

## 基本信息

| 属性 | 值 |
|------|-----|
| 爆发日期 | 1861年4月15日 |
| 结束日期 | 1865年4月9日 |
| 交战双方 | 北方联邦（美利坚合众国）vs 南方邦联（美利坚联盟国） |
| 起因 | 奴隶制存废争议、州权之争 |
| 结果 | 北方胜利，奴隶制废除 |
| 死亡人数 | 约62万人 |

## 事件概述

1861年4月15日，林肯政府发布讨伐令，向南部联盟宣战，美国南北战争正式爆发。

战争根源在于南北双方在奴隶制问题上的根本分歧。南方11个蓄奴州宣布脱离联邦，成立"美利坚联盟国"。林肯就任后拒绝承认南方独立，双方兵戎相见。

战争持续4年，1865年4月9日，南方邦联将军罗伯特·李向北方格兰特将军投降，战争结束。

## 历史意义

- **奴隶制终结**：《解放黑人奴隶宣言》使400万黑奴获得自由
- **联邦统一**：确立了联邦不可分裂的原则
- **美国转型**：战争摧毁了种植园经济，为美国工业化铺平道路
- **宪法修正**：通过第13、14、15修正案，废除奴隶制、确立公民权利
