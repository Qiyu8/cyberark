# 因果报应

## 基本信息

- **梵文**：Karma（业），因果报应即 Karma Phala
- **地位**：佛教因果律的核心，整个佛教伦理的基础
- **经典依据**：《四十二章经》、《缘起经》、《十善业道经》等

## 核心原理

### 业力法则

"业"（Karma）原意为"行为"，在佛教中指身、口、意三业所造之因：

- **身业**：杀、盗、淫
- **口业**：妄语、两舌、恶口、绮语
- **意业**：贪、嗔、痴

> 善业得乐报，恶业得苦报，如影随形，丝毫不爽。

### 因果三定律

1. **自作业**：自作自受，业因不能转嫁他人
2. **不失坏**：业因未感果报前，永不消失
3. **遍因果**：善恶业因必感相应果报

### 业的分类

| 分类标准 | 类型 |
|---------|------|
| 道德属性 | 善业、恶业、无记业 |
| 感果时间 | 现法受业、次生受业、后后受业 |
| 受报次数 | 决定业、不决定业 |

## 业报轮回

### 六道轮回

| 道 | 报应 | 主要业因 |
|----|------|----------|
| 天道 | 享天福 | 十善圆满 + 禅定 |
| 人道 | 苦乐参半 | 修五戒 |
| 阿修罗道 | 斗争不断 | 嫉妒、斗争 |
| 畜生道 | 被驱役 | 愚痴、偷盗 |
| 饿鬼道 | 饥渴常乏 | 悭贪 |
| 地狱道 | 剧苦无间 | 十恶重罪 |

### 业力与轮回

- 不是灵魂在轮回，而是业习在相续
- 中阴身在业力推动下投生六道
- 一生所造业因决定来世去向

## 现世因果

### 善有善报

| 善业 | 乐报 |
|------|------|
| 不杀生 | 长寿、无病 |
| 不偷盗 | 财富丰饶 |
| 不邪淫 | 眷属清净 |
| 不妄语 | 得人信任 |
| 不饮酒 | 智慧清明 |

### 恶有恶报

| 恶业 | 苦报 |
|------|------|
| 杀生 | 短命、多病 |
| 偷盗 | 贫穷、损耗 |
| 邪淫 | 眷属不贞 |
| 妄语 | 不被信任 |
| 饮酒 | 愚痴颠倒 |

## 因果的超越

大乘佛教认为因果法则虽不可违，但通过忏悔、修行可以：
- **转重为轻**：重业通过修行转为轻受
- **转染成净**：以智慧照破无明，不再造染污业
- **了因即果**：彻见缘起性空，因果如梦幻泡影
