# 地藏经

## 基本信息

- **全称**：《地藏菩萨本愿经》
- **译者**：实叉难陀译（唐）
- **地位**：中国民间流传最广的佛经之一，佛教孝经
- **篇幅**：上下两卷，十三品

## 核心内容

### 经文结构

| 品名 | 核心内容 |
|------|----------|
| 忉利天宫神通品 | 释迦牟尼于忉利天为母说法，地藏菩萨因地发愿 |
| 分身集会品 | 十方世界地藏菩萨分身集会 |
| 观众生业缘品 | 众生业缘差别，无间地狱情形 |
| 阎浮众生业感品 | 地藏菩萨因地所发二愿 |
| 地狱名号品 | 详述各种地狱名称及罪报 |
| 如来赞叹品 | 佛陀称扬地藏菩萨利益人天 |
| 利益存亡品 | 存亡获利之事 |
| 阎罗王众赞叹品 | 鬼王菩萨赞叹地藏功德 |
| 称佛名号品 | 佛说过去佛名号功德 |
| 校量布施功德缘品 | 布施因缘果报 |
| 地神护法品 | 坚牢地神护持供养地藏者 |
| 见闻利益品 | 观世音菩萨与地藏菩萨利益众生 |
| 嘱累人天品 | 释迦牟尼将娑婆众生托付地藏 |

### 核心主题

地藏菩萨"地狱不空，誓不成佛"的宏愿，是全经核心。经中详述因果报应、业力法则，对中国民间信仰影响深远。

## 著名偈语

> "我今尽未来际，不可计劫，为是罪苦六道众生，广设方便，尽令解脱，而我自身，方成佛道。"

## 核心概念

### 无间地狱（阿鼻地狱）

经中详细描述无间地狱五无间：
- **时无间**：日月流转，罪苦不断
- **形无间**：罪人遍满地狱，身体亦无间
- **果无间**：罪器叉棒，满在身上
- **命无间**：罪人断命续之苦无间
- **趣无间**：随其业感，遍入此趣

### 婆罗门女/光目女

地藏菩萨因地为婆罗门女、光目女时，因救度母亲而发下大愿。此故事成为中国"孝道"文化的佛教典范。

## 影响与意义

- **孝道典范**："目莲救母"的故事源自地藏经，是中国古代孝道文学的母题
- **超度亡魂**：地藏经被认为是超度亡灵最有效的经典，在中国丧葬文化中地位极高
- **地藏信仰**：九华山是地藏菩萨道场，每年农历七月三十为地藏菩萨圣诞，香火鼎盛
- **民间法事**："打地藏七"、诵地藏经超度成为汉地佛教最普遍的法事活动
