# 法华经

## 基本信息

- **全称**：《妙法莲华经》
- **译者**：后秦·鸠摩罗什译
- **地位**：大乘佛教最重要的经典之一，天台宗立宗之本经
- **篇幅**：七卷二十八品，约六万字

## 核心内容

### 核心主题：一佛乘

《法华经》最核心的思想是"会三归一"——将声闻乘、缘觉乘、菩萨乘三乘方便法门，会归到唯一佛乘。一切众生皆可成佛。

### 著名品目

| 品名 | 核心内容 |
|------|----------|
| 方便品 | 十方佛土中，唯有一乘法；诸佛出世大事因缘 |
| 譬喻品 | 火宅喻：三界火宅，唯有一乘大白牛车能救人 |
| 信解品 | 穷子喻：众生如失散之子，终将回归本有家财 |
| 药草喻品 | 三草二木喻：随类受化，皆能成佛 |
| 授记品 | 为四大弟子等授记成佛 |
| 化城喻品 | 化城喻：引导众生渐次修行至宝所 |
| 观世音菩萨普门品 | 观音菩萨寻声救苦，广大灵感 |
| 普贤菩萨劝发品 | 普贤菩萨守护持经者 |

### 著名偈语

> "诸法从本来，常自寂灭相。"

> "佛种从缘起，是故说一乘。"

## 核心概念

### 三乘归一

- **声闻乘**：听闻佛陀说法而悟道，证阿罗汉果
- **缘觉乘**：观十二因缘而悟道，证辟支佛果
- **菩萨乘**：发菩提心，修六度万行，志在成佛

《法华经》指出：这三乘只是方便引导，最终目的都是令众生开示悟入佛之知见，证得无上菩提。

### 久远实成

《法华经》宣说释迦牟尼并非仅在印度成佛，而是久远劫前已成佛，此娑婆世界不过是方便示现。此说对后世佛教思想影响深远。

## 影响与意义

- **天台宗**：智者大师依《法华经》创立天台宗，是中国第一个真正意义上的大乘佛教宗派
- **观音信仰**：《普门品》使观世音菩萨信仰在中国民间极度兴盛
- **文学价值**：经中大量生动譬喻（火宅喻、穷子喻、化城喻、衣珠喻等）成为中国文学母题
- **艺术宝库**：敦煌、云冈、龙门石窟中，法华经变相是最重要的创作题材之一
