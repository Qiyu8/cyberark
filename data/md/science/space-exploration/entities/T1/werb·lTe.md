# 维尔伯·莱特

## 基本信息

- **出生**: 1867年4月16日
- **逝世**: 1912年5月30日（享年45岁）
- **国籍**: 美国
- **职业**: 发明家、机械师、航空先驱

## 历史贡献

维尔伯·莱特与弟弟奥维尔·莱特（莱特兄弟）共同发明了世界上第一架实用飞机，实现了人类首次动力飞行。

### 首次动力飞行

| 项目 | 内容 |
|------|------|
| 日期 | 1903年12月17日 |
| 地点 | 美国北卡罗来纳州基蒂霍克 |
| 飞机 | 飞行者一号（Wright Flyer） |
| 飞行者 | 奥维尔·莱特（首次飞行） |
| 飞行距离 | 36.5米（12秒） |
| 当天最长 | 260米（59秒，维尔伯驾驶） |

### 技术突破

1. **三轴控制系统**
   - 偏航控制（方向舵）
   - 俯仰控制（升降舵）
   - 滚转控制（机翼扭曲）
   - 现代飞机控制的基础

2. **风洞实验**
   - 自制风洞测试机翼形状
   - 积累大量空气动力学数据

3. **轻型发动机**
   - 与机械师合作设计
   - 重量仅约80公斤
   - 功率约12马力

## 历史影响

1. **航空时代开启**：人类进入动力飞行时代
2. **军事革命**：改变了战争形态
3. **全球化加速**：缩短了世界各地的距离
4. **经济变革**：催生了航空工业和民航业

## 维尔伯的贡献

- 主要负责理论研究和飞行测试
- 1908年在法国进行公开飞行表演，震惊欧洲
- 1909年获法国荣誉军团勋章
- 与奥维尔共同获得多项航空专利
