## 生平简介

尤里·布朗芬布伦纳（Urie Bronfenbrenner，1917年4月29日—2005年9月25日），美国发展心理学家，以其生态系统理论（Ecological Systems Theory）闻名于世，彻底改变了我们对儿童发展环境的理解。

布朗芬布伦纳出生于俄罗斯莫斯科的一个犹太家庭。1919年随家人移居美国，在明尼苏达州长大。1938年进入新泽西州拉斐特学院学习，1939年获得学士学位。

1942年获得康奈尔大学硕士学位，1948年获得博士学位。此后一直在康奈尔大学任教，直到1994年退休。

## 主要理论与贡献

### 生态系统理论

布朗芬布伦纳提出的人类发展生态系统理论是其最著名的贡献：

**嵌套结构模型：**

```
          ┌─────────────────────┐
          │    外系统           │
          │  ┌───────────────┐ │
          │  │   中系统       │ │
          │  │  ┌───────────┐ │ │
          │  │  │  微系统   │ │ │
          │  │  └───────────┘ │ │
          │  └───────────────┘ │
          └─────────────────────┘
                  ┌─────────────────────┐
                  │   大系统            │
                  └─────────────────────┘
```

**五层系统：**

1. **微系统（Microsystem）**
   - 儿童直接接触的环境
   - 家庭、学校、同伴群体

2. **中系统（Mesosystem）**
   - 微系统之间的相互联系
   - 家庭与学校的联系

3. **外系统（Exosystem）**
   - 儿童间接接触的环境
   - 父母的工作场所、社区

4. **大系统（Macrosystem）**
   - 文化和社会价值观
   - 经济制度、政治体系

5. **时间系统（Chronosystem）**
   - 时间维度
   - 历史时期和生活事件

## 影响与遗产

布朗芬布伦纳的理论对多个领域产生了深远影响：

- **发展心理学**：改变了儿童发展研究的视角
- **教育**：影响了学校和家庭合作
- **政策制定**：影响了儿童和家庭政策
- **公共健康**：影响了健康促进项目
- **跨文化研究**：促进了文化敏感性
